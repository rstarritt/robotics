2.1 安装¶

系统要求¶

Windows¶

  • Microsoft® Windows® 10/8.1/8/7 (32 or 64-bit)

  • 150 MB 磁盘空间

  • USB 2.0 接口

  • Microsoft® Visual Studio 2013 Community Edition 版本或更高

Mac OS X¶

  • Mac® OS X® 10.8.5 版本或更高

  • 150 MB 磁盘空间

  • USB 2.0 接口

Android¶

  • Version 4.4 (Lollipop) 版本或更高

  • ARM
  • USB 2 OTG

相机要求¶

  • Astra

安装SDK¶

安装 Astra SDK前，请先到 这里 下载最新版本SDK。下载完成后，请将SDK解压到您可以记住的位置。

编译示例程序¶

Windows¶

  1. 在已解压的SDK路径的/samples/vs2013 或 /samples/vs2015 目录下找到 Astra-samples.sln

  2. 用Visual Studio 2013 或 Visual Studio 2015打开对应的 Astra-samples.sln 文件

  3. 在Visual Studio中通过菜单中的 BUILD -> Build Solution 或按快捷键 Ctrl+Shift+B 来编译示例程序

  4. 稍等片刻，编译完成后程序即可开始运行

  5. 如需在Visual Studio中启动示例程序，可以右键点击待启动的工程文件，然后选择‘Set as StartUp Project’，就可以将该示例程序设置为启动项目。之后通过使用菜单中的 DEBUG -> Start Debugging 或点击键盘上的 F5 键来进行debug

OS X¶

示例程序使用CMake 3.2+进行编译

获取CMake¶

  • 方法 1: 从以下地址下载 https://cmake.org/download/

  • 方法 2: 如果您是homebrew(https://brew.sh)用户，可以在terminal中使用以下方式安装最新的cmake:

    $ brew install cmake

编译例子¶

  1. 打开terminal然后将路径切到SDK中的 samples/ 文件夹

  2. 输入 ./build_samples.sh

  3. 示例程序即被编译完成至 samples/build/bin/ 路径，您就可以在terminal中运行示例程序了，例如:

    $ cd build/bin
$ ./SimpleStreamViewer-SFML
